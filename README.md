# odin-recipes

A recipe website with a collection of simple and budget friendly recipes for people who are not that confident with cooking 

## Description
This is website has a collection of recipes that has a separate page for each recipe that is both accessible and responsive

The main aim for this page is to help people who might not be confident in their cooking skills or don't know how to cook but have found themselves in a position where they have to cook, whether they are just beginning to live on their own or need to impress someone

The recipes are simple to follow and the ingredients are budget friendly to accommodate anyone no matter the skill level

## Features
- Accessible
- Fully Responsive
- Flexbox and Grid layout structure
- Navigation bar (responsive)
- Clean and modern UI layout

## Stack
- HTML
- CSS

## Installation

1. Download or clone the repository
2. Open project folder
3. Open `index.html` in your browser
